Admin-Invited Provider Onboarding
Complete workflow for admins to create and invite providers with rep attribution

Key Features:

  • Admin creates provider profile with required information (NPI, clinic details)
  • Rep assignment is mandatory during creation
  • System validates NPI format before proceeding
  • Automated invite email sent with unique activation link
  • Provider sets password and activates account
  • Rep attribution is permanent and automatic for all future orders

Public Provider Application & Approval
Self-service application flow with admin review and approval workflow

Key Features:

  • Provider-initiated application process via public form
  • Document uploads required (business license, reseller certificate)
  • Admin receives notification of new pending applications
  • Admin reviews documents and assigns rep before approval
  • Rejection requires reason (sent to provider via email)
  • Approval triggers automated welcome email with setup link
  • Optional rep assignment (can be marked as "Unassigned")
